FireEdit

A simple light weight js editor
FireEdit is a js editor based on ACE and narcissus. It runs as a plugin in firefox 7.x and above. user ctrl-shift-f to activate the editor or "FireEdit" from page context menu, this will open a new tab with the editor. Ignore the url displayed in the editor, it points to the addon url on the local machine. Currently works with Firefox 7 and above.

Feature List :
  1. Work with local files (completed)
  2. Method window
  3. Refactoring
    1. Method rename
    2. Variable rename
    3. Extract Methods
  4. Key customization support
  5. Custom save actions
  6. Debugging
